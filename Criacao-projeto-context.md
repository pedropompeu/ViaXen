# 🧬 DNA do Projeto - RotaCerta (Edição Orçamento Zero)

**Data de Criação:** 23/05/2026
**Status:** Planejamento Estratégico (C-Level)

## 🎯 Visão do Produto
Sistema de planejamento logístico e cálculo de fretes otimizado para o mercado brasileiro, construído inteiramente com tecnologias Open Source e Gratuitas.

## 🏛️ Stack Tecnológica (100% FREE)
- **Frontend:** React + Tailwind CSS (Vite).
- **Mapas & Rotas:** Leaflet.js + OpenStreetMap (Grátis) + OSRM (Open Source Routing Machine).
- **Big Data / Analytics:** DuckDB (Local, rápido e grátis) + Arquivos Parquet.
- **Visualização:** Power BI Desktop (Versão Gratuita) + Exportação de dados via Python/DuckDB.
- **Geocodificação:** Nominatim (OpenStreetMap) com limites de requisição respeitados.
- **Hospedagem:** Vercel / Netlify (Tier Gratuito).
- **Banco de Dados:** Supabase (Tier Gratuito) ou LocalStorage/IndexedDB para protótipo.

## 💰 Modelo de Negócio (ROI)
- Custo de infraestrutura: R$ 0,00.
- Valor gerado: Economia em APIs de mapas proprietárias e otimização de combustível.

## 🗺️ Roadmap MVP
1. Setup Monorepo (Free Stack).
